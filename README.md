## intro

本仓库是一个简单的分布式系统的golang实现，来源于b站up主[软件工艺师](https://space.bilibili.com/361469957)

## 系统组件

* 服务注册：服务注册，健康检查

* 用户门户：web应用，api网关

* 日志服务：集中式日志

* 业务服务：业务逻辑，数据持久化

## 技术选型

* 开发语言：go
* 数据传输：HTTP
* 传输协议：JSON
